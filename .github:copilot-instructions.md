# üîå IMPLEMENTATION

## Detect iFrame

```js
export const isEmbedded = () => window.self !== window.top;
```

---

## Auth Hook

```js
useEffect(() => {
  if (isEmbedded()) {
    window.parent.postMessage({ type: 'REQUEST_AUTH_TOKEN' }, '*');
  }
}, []);
```

---

## Handle Token

```js
window.addEventListener('message', (event) => {
  if (event.origin !== 'https://kumii.africa') return;
  if (event.data.type === 'KUMII_AUTH_TOKEN') {
    setToken(event.data.token);
  }
});
```

---

## Send Events

```js
window.parent.postMessage({ type: 'COURSE_COMPLETED', courseId }, 'https://kumii.africa');
```
